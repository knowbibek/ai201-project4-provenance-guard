# Provenance Guard

A backend service any creative-sharing platform can plug into to classify whether submitted
text was likely written by a human or generated by AI — then surface an honest, plain-language
transparency label and let creators appeal a classification.

> **Status:** Milestone 0 — project scaffolded. Implementation not started yet.
> Design lives in [planning.md](planning.md).

## Tech Stack

| Component          | Tool                                  |
|--------------------|---------------------------------------|
| API framework      | Flask                                 |
| Detection signal 1 | Groq (`llama-3.3-70b-versatile`)      |
| Detection signal 2 | Stylometric heuristics (pure Python)  |
| Rate limiting      | Flask-Limiter                         |
| Audit log          | SQLite                                |

## Setup

```bash
# clone, then from the project root:
python -m venv .venv
source .venv/bin/activate          # Mac/Linux
# .venv\Scripts\activate           # Windows

pip install -r requirements.txt

cp .env.example .env               # then add your GROQ_API_KEY
```
